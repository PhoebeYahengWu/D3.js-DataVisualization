Top 10 Most Popular Programming Languages

by Phoebe Yaheng Wu

Programming is one of the essential skills required by many well-paid jobs. Besides, programming skills enable people to make a positive impact in many ways either in their jobs, industry or society, for example, by helping with efficiency, accessibility and automation. However, since the introduction of programming, there have been over 700 notable programming languages according to Wikipedia. That is indeed overwhelming. But it isn't necessary to learn them all. In fact, if one programming language enables you to do what you want to do, you can start with that and master it gradually. So, which programming language should you choose? I believe the 2019 Stack Overflow survey responded to by 87354 developers can be a good reference to begin with.

According to the 2019 StackOverflow developer survey, JavaScript is the most commonly used programming language for the seventh year in a row.

Below are the top 10 most popular programming, scripting, and markup languages according to the 2019 Stack Overflow survey. As we can see, JavaScript is the most popular programming language favored by more than 60% of developers. It is followed by markup language HTML and styling language CSS. The query language SQL ranks third the most popular. Notably, the popularity of Python has outshined Java, C#, PHP, C++ and C.

When exactly were these top 10 popular programming languages introduced?

As we can see from the above timeline, a major language comes out every few years, not even including the 300 other languages. The industry changes so fast with the release of every new programming language. However, each programming language serves a different purpose. For example, JavaScript can help web developer building dynamic and interactive websites, while Python can help data scientist to process and analyze big data. You need to know what you want to do first then choose the programming language that helps you achieve it. Below is the list of different options for you to reference:

• Front-end web development – JavaScript, HTML/CSS
• Back-end web development – JavaScript, Python, Java, PHP, C#
• Desktop application – Java, C++, Python
• Game development – C++, C#
• Data science and Machine learning – Python
• Database – SQL

All in all, the logic behind each programming language is almost the same. If you can master one programming language, it's quite easy to move on to the second one. What we should focus on is determining the most suitable route to begin with and then finding the most used programming languages in that route. Finally, keep practicing!

Data Source: 2019 Stack Overflow Developer Survey
